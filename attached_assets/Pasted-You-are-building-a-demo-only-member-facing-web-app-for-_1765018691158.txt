You are building a demo-only, member-facing web app for Airborne Fitness, an aerial fitness studio in Mumbai.

The goal is to create a polished, mobile-first, highly aesthetic front-end that simulates all flows.
No real backend integration is required – mock data and fake APIs are fine.
Focus on UI, UX, flows and state management, not production backend.

1. Tech Stack & Project Setup

Use Next.js (latest) with TypeScript.

Use App Router.

Use Tailwind CSS for styling.

Assume Firebase Firestore and Razorpay exist conceptually, but:

For this demo, do NOT actually connect to Firebase or Razorpay.

Instead, simulate network calls with mock async functions and local state (React context or simple in-memory store).

No need to actually send SMS – simulate OTP flow on the frontend.

Project structure suggestion:

/app

/login

/dashboard

/enroll

/book

/sessions

/news

/components

/lib (for mock data + utilities)

/styles
